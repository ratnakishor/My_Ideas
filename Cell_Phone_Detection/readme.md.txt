# Cell Phone Detection Using YOLO model.

System detects the presence of a cell phone using YOLO model and sends that image in to a WhatsApp number.

We need to open WhatsApp in the default web browser.

To install dependencies use "pip install requirements.txt" from the command prompt.

To run the application use the receiver phone number in send_message function and use "python CellPhoneDetection.py" from the command prompt.
